# 🤝 Guía para Contribuir a LEGADO

¡Gracias por tu interés en LEGADO! Toda ayuda es bienvenida. Aquí te explicamos cómo puedes contribuir.

## 📋 Formas de contribuir

### 1. Reportar errores (Issues)
Si encuentras un bug, abre un **Issue** con:
- Descripción clara del problema
- Pasos para reproducirlo
- Captura de pantalla (si aplica)
- Etiqueta `bug`

### 2. Sugerir mejoras
¿Tienes una idea? Abre un **Issue** con:
- Descripción de la mejora
- Por qué sería útil
- Etiqueta `enhancement`

### 3. Escribir código
1. Haz **Fork** del repositorio
2. Clona tu fork: `git clone https://github.com/tu-usuario/legado-sst-core`
3. Crea una rama: `git checkout -b mi-nueva-funcion`
4. Haz tus cambios y haz commit: `git commit -m "Descripción clara"`
5. Sube tus cambios: `git push origin mi-nueva-funcion`
6. Abre un **Pull Request** hacia `main`

## 📝 Estilo de código
- Usa nombres descriptivos
- Comenta en español o inglés (consistente)
- Respeta el formato existente

## 🧪 Pruebas
Si agregas funcionalidad nueva, asegúrate de probarla antes de enviar el PR.

## 🏛️ Filosofía LEGADO
- **Simplicidad**: Código limpio y fácil de entender
- **Seguridad**: Priorizar la protección de datos
- **Accesibilidad**: Que cualquier PYME pueda usarlo

## 📧 Contacto
¿Dudas? Abre un Issue o contacta al mantenedor.
