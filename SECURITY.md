# 🔒 Política de Seguridad de LEGADO

## 📢 Reportar una vulnerabilidad

**No reportes vulnerabilidades en Issues públicos.**

Envíanos un reporte a través de:
- **GitHub Security Advisory**: Ve a la pestaña "Security" → "Report a vulnerability"
- **Email**: jagc_2011@hotmail.com

## ⏱️ Tiempos de respuesta

| Tipo de reporte | Tiempo de respuesta |
|-----------------|---------------------|
| Crítica | 24 horas |
| Alta | 48 horas |
| Media | 72 horas |
| Baja | 1 semana |

## ✅ Buenas prácticas
- No subas claves API al repositorio
- Usa variables de entorno para datos sensibles
- Mantén las dependencias actualizadas

**La seguridad es prioridad en LEGADO.** 🛡️
